# Previsão de Diabetes com Naive Bayes

Projeto completo de machine learning para previsão de diabetes utilizando o algoritmo Naive Bayes Gaussiano, com análise exploratória, pré-processamento e avaliação de modelo.

##  Sobre o Projeto

Este projeto implementa um classificador para prever diabetes com base em duas características clínicas:
- Níveis de glucose
- Pressão arterial

##  Funcionalidades

- **Análise Exploratória Completa:**
  - Visualização de dados com head(), tail(), info()
  - Estatísticas descritivas com describe()
  - Verificação de valores ausentes

- **Pré-processamento Avançado:**
  - Normalização Min-Max (escala 0-1)
  - Separação estratégica de dados (treino/teste)

- **Modelagem Preditiva:**
  - Implementação do algoritmo Naive Bayes Gaussiano
  - Treinamento e previsão com dados reais

- **Avaliação do Modelo:**
  - Matriz de confusão visual
  - Cálculo de acurácia
  - Análise de overfitting
